# healthyR.ts Wiki

Welcome to the comprehensive wiki for the **healthyR.ts** R package!

## About This Wiki

This wiki provides in-depth documentation, tutorials, and guides for using healthyR.ts for time series analysis and forecasting of hospital administrative and clinical data.
